# Compassion International Icon Font

## Loading into IcoMoon
You can import *selection.json* back to the IcoMoon app (https://icomoon.io) using the *Import Icons* button (or via Main Menu â†’ Manage Projects) to retrieve your icon selection. 

## Building a .woff2 file
To build that you need to use something like https://transfonter.org/

- On transfonter you click "Add fonts" and choose the icomoon woff file.
- Leave selections at their default (making sure woff2 is selected in Formats).
- Click the "Convert" button.
- Download the provided zip
